Identifiers - An identifier is a sequence of characters in the code that Identifiers
            a variable, function or a property.
    
    Rules while creating Identifiers
        1) It should not be a reserved word.
        2) it should not start with a number.
        3) it should not contain any special character other than underscore(_)
            and dollar($).
        4) Javascript variable names are case-sensitive.
