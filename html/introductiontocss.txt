what is Css?
	css stands for Cascading style sheet, which helps developers to apply styles to html elements within the page.

History of CSS
	- css 1.0
	- css 2.0
	- css 3.0

How are we going to apply styles to elements?
	- Inline
	- Internal 
	- External 

what are the core concepts of css?
	- Selectors - Which helps developers to select an element.
	- Properties - which hepps developers to apply properties to an element for styling.
